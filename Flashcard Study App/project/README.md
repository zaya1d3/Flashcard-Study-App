# Flashcard App
#### Video Demo: <https://youtu.be/vWrg43jXKdo>
#### Description:
Flashcards by Zayan is an interactive web application designed to help users efficiently study and retain information using digital flashcards. This project leverages Python, Flask, SQL, HTML, CSS, and JavaScript, incorporating the skills learned throughout CS50x. The application offers a complete end-to-end experience for managing and studying flashcards in a user-friendly and visually appealing way.

Features

User Authentication

Users can securely register and log in using unique usernames and hashed passwords.

Registration ensures that multiple users can maintain their own personal flashcard decks.

Flashcard Management

Users can add new flashcards with questions and answers.

Edit existing flashcards to update information as needed.

Delete flashcards that are no longer relevant.

All flashcards are stored in a SQLite database for easy retrieval and persistence.

Study Mode

Users can study their flashcards in an interactive 3D flip card format, making memorization more engaging.

Shuffle cards to randomize the order, helping improve recall and learning.

Navigation arrows allow users to easily move forward or backward between cards.

Progress Tracking

A progress bar visually displays how many flashcards have been studied and how many remain, motivating users to complete sessions.

User Experience Enhancements

Dark mode toggle for comfortable use in different lighting conditions.

Smooth animations and responsive design ensure that the app works well on desktop and mobile devices.

Technical Overview

Backend: Python with Flask framework handles routing, user authentication, and database operations.

Database: SQLite manages user data and flashcards efficiently.

Frontend: HTML, CSS, and JavaScript create an interactive, visually appealing interface.

Security: Passwords are hashed before storage to protect user accounts.

Development Notes

The app demonstrates the integration of Flask routing, SQL queries, and dynamic HTML rendering.

JavaScript is used for flashcard flipping, shuffling, and progress bar updates in real time.

Design decisions, like 3D flip cards and dark mode, were included to make studying engaging and user-friendly.

Why I Built This

The goal of this project was to create a practical study tool that showcases my CS50x skills. Flashcards are a proven method for learning, and by creating a digital version, I wanted to combine usability, functionality, and visual appeal. This app could be used by students, professionals, or anyone looking to improve memory retention in an interactive way.

Future Improvements

User accounts could have multiple decks or categories for better organization.

Add a quiz mode that tests knowledge based on flashcards studied.

Include statistics tracking to show how well a user remembers content over time.

Flashcards by Zayan is a complete, interactive, and secure application that turns studying into a fun and effective experience.
